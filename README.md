# Exihibbbit

Exihibbbit is a Dribbble.com Android client, it’s an online community for designers to share, discuss their work.

## Features

Shots:
• Like/unlike a shot

• See likes count

• See views count

• Watch GIFs

• Share shots


Buckets:
• See your buckets

• Create new bucket

• Add a shot to a bucket


Users:
• Login/Logout
